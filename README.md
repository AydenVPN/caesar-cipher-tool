# Caesar Cipher Tool

A simple Python-based Caesar Cipher encryption and decryption tool that demonstrates basic cryptography concepts and ASCII character manipulation.

---

## 🔐 Features

- Encrypt messages
- Decrypt messages
- Custom shift keys
- Preserves spaces and symbols
- Command-line interface (CLI)

---

## 🧠 How It Works

The Caesar Cipher shifts letters in the alphabet by a specified number.

Example:

Plaintext: `HELLO`  
Shift Key: `3`  
Ciphertext: `KHOOR`

---

## 🚀 How to Run

1. Open a terminal in the project folder
2. Run:

```bash
python3 main.py
```

---

## 💡 Example Usage

```text
Would you like to encrypt or decrypt a message? E/D: E
Message to Encrypt: HELLO
Shift key (Integer): 3

Encrypted Message: KHOOR
```
